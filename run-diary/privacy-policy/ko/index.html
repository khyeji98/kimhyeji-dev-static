개인정보처리방침

'달림일기'(이하 '앱')는 사용자의 개인정보를 매우 중요시하며, 「개인정보보호법」을 준수하고 있습니다. 본 앱은 별도의 서버를 운영하지 않으며, 개발자가 사용자의 개인정보를 수집하거나 가져가지 않습니다.

1. 개인정보의 처리 및 저장 방식

본 앱은 회원가입이나 로그인을 요구하지 않습니다. 사용자가 입력한 데이터와 건강 데이터는 사용자 본인의 휴대전화 내부 저장소에만 암호화되어 저장됩니다. 개발자는 이 데이터에 접근할 수 없으며, 별도의 외부 서버로 전송되지 않습니다.

2. 처리하는 개인정보 항목 및 목적

앱 기능 구현을 위해 기기 내부에서만 다음과 같은 정보를 처리합니다.

  • 건강 데이터(HealthKit): 사용자의 동의 하에 Apple '건강' 앱으로부터 걸음 수, 심박수, 소모 칼로리, 운동 거리 데이터를 가져와 앱 화면에 표시하고 통계를 제공하는 목적으로만 사용합니다.
  • 사용자 입력 정보: 사용자가 직접 작성한 러닝 메모, 컨디션, 착용 신발 등의 기록.

3. HealthKit 데이터의 보호

Apple 건강(HealthKit) 데이터와 관련하여 다음 사항을 준수합니다.

  • 건강 데이터를 마케팅, 광고, 데이터 분석 목적으로 절대 사용하지 않습니다.
  • 사용자의 명시적인 동의 없이는 건강 데이터를 제3자와 공유하지 않습니다.
  • 건강 데이터는 오직 사용자의 기기 내에서만 처리되며 외부로 유출되지 않습니다.

4. 개인정보의 파기

본 앱은 서버에 데이터를 저장하지 않으므로, 사용자가 앱을 삭제하는 경우 기기에 저장된 모든 데이터는 즉시 삭제되어 복구할 수 없습니다.

5. 개인정보 보호책임자
